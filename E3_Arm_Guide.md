# **E3 Arm UI** Guide

## Prerequisites:

1. Robot is assembled
1. SD card is in the Raspberry Pi on the robot
1. Batteries are charged and installed
1. Pi500 is connected to the robot using VNC 

**Arm Interface**

   python /home/pi/MasterPi_PC_Software/Arm.py
